Sample programs found in the examples directory

Sample1: Simple demonstration of video capture using the libARvideo.lib library

Sample2: Simple demonstration of code that detects the tracking markers

Sample3: Simple code for recognizing the pattern inside a tracking marker

Sample4: Code which shows OpenGL graphic overlay on a tracking marker

simple: Code showing dynamic changing of threshold values and tracking modes

multiTest: Example of tracking from multiple markers

Exview: Tracking utility showing camera pose calculations

calib_camera2: Camera calibration software

mk_patt: Pattern training software

simpleVRML: Simple ARToolKit application that loads a VRML file
